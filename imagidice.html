About: Embarking on an exciting hobby project, my brother and I have set out to create a web-based version of the imaginative board game known as Imagidice. This unique game involves players rolling six-sided dice, with each die displaying different images that serve as prompts to collaboratively craft a captivating story. As the project unfolds, I've been immersed in shaping the front-end design, while my brother dedicates his efforts to coding the dynamic back-end of the website. Although a work in progress, this endeavor is a source of pride for me.
